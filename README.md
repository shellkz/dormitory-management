# Requirements

| Python | Version |
|--------|---------|
| Minimum | 3.10+ |
| Tested | 3.14 |

# Run
1. Windows
```bash
git clone https://github.com/shellkz/dormitory-management.git
cd dormitory-management
python -m venv venv
venv\Scripts\activate        
pip install -r requirements.txt
python main.py
```

2. Linux and Mac
```bash
git clone https://github.com/shellkz/dormitory-management.git
cd dormitory-management
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python main.py
```
